# Authority Management-System


## 当前版本：v1.3.1

## Install
```bush
// install dependencies
npm install
```
## Run
### Development
```bush
npm run dev
```
### Production(Build)
```bush
npm run build
```

## 简介
&emsp;&emsp;Management-System是基于iView admin，搭配使用[iView](https://www.iviewui.com) UI组件库形成的一套后台管理系统

## 功能
- 登录/登出
- 用户管理
- 角色管理

## 文件结构
```shell
.
├── build  项目构建配置
└── src
    ├── images  图片文件
    ├── libs  工具方法
    ├── router  路由配置
    ├── store  状态管理
    ├── styles  样式文件
    ├── template  模板文件
    ├── vendors  公共库文件
    └── views

```
